Game utilities for HADES`STAR

HSM (Hades`Star acount manager )  
version 0.12
Application for quick launch of the selected game account. Relevant for those who have two or more accounts.
Running one account after leaving the game, you can choose another account from a pre-prepared list. To use the utility, you should save your accounts that will be used in any folder on the device in a convenient location. (To do this, start the game with this account and then save the file login.info in a convenient place of the device. After that, in the utility settings specify the location of the login file (no matter how the file will be called and its extension), specify the root folder where the game. The program will determine whether the file meets the login, whether the specified directory of the game.
Specifying the game directory in this case is done manually in order to specify the original game or mod. (they are located in different folders and have different names)
After the settings are made, you can start the game with any of these entered accounts, which allows you to change them without resorting to Google accounts, second spaces on the device and other difficulties.
In the application there is no advertising.  use (video)


ConnectHS
Software, independent analogue of PRSS
(the application uses FB in its free part, so the number of simultaneous connections is limited in time and the application is reduced to 20 minutes when forming a command)

App ConnectHS was created to carry out synchronous start command typed in Red Stars on all or selected levels.
According to the time report you should click scan. The countdown at all is performed synchronously. The application itself will not do it for you, but will indicate this message at what point to do it.
With the ConnectHS app :
- you can form a team and continue to play - in case of joining players will receive a message in the game and a notification(optional);
- you can remove players from the team;
- you can close the set of the team - in such cases, no one else will join you;
- you can correspond in the team chat;
- optional countdown to the start of the spoken voice (if you have installed on the device audio package);
- report start and modify command from the application you get are always from wherever you currently were not(in other chats or in-game)
- you can disable / enable the automatic start of the game after the start.
In the application there is no advertising.
